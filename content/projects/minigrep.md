A minimal grep implementation in Rust focusing on speed and simplicity.

## Features

- Pattern matching with basic regex
- Recursive directory search
- Colored output for matches
- Memory-mapped file I/O for performance

## Performance

Searches 1GB of text files in ~200ms (compared to GNU grep's ~180ms).

## Why Another Grep?

Educational project to learn:
- Systems programming in Rust
- Efficient file I/O patterns
- Regex engine implementation
